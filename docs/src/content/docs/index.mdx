## Quick Start

Install excel-to-json and start converting files in seconds:

<Tabs>
	<TabItem label="macOS/Linux">
		```bash
		# Install with cargo
		cargo install excel-to-json

		# Convert an Excel file
		excel-to-json data.xlsx
		```
	</TabItem>
	<TabItem label="Windows">
		```powershell
		# Install with cargo
		cargo install excel-to-json

		# Convert an Excel file
		excel-to-json.exe data.xlsx
		```
	</TabItem>
</Tabs>
